# Contributing to Aurelius

Thank you for your interest in contributing to Aurelius!
</div>

## Have a Bug or Idea?

**Please open an issue!** This is the best way to report bugs or suggest features.

When creating an issue, please include:
- **For bugs**: Clear description of the problem, steps to reproduce, and expected vs. actual behavior.
- **For ideas/features**: Detailed description of the proposed enhancement and its benefits.

(no pressure)

### Types of Contributions

- **Bug fixes**: Improve reliability and correctness
- **Feature requests**: Suggest new capabilities or optimizations
- **Performance improvements**: Enhance search speed or evaluation accuracy
- **Documentation**: Improve README, comments, or code clarity
- **Testing**: Help identify edge cases and verify functionality

## Questions?

Open an issue— I'm here to help!
